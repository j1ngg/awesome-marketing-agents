# Contributing Guidelines

Please note that this project is released with a [Contributor Code of Conduct](code-of-conduct.md). By participating in this project you agree to abide by its terms.

## Adding a New Resource

Please ensure your pull request adheres to the following guidelines:

- **Relevance:** The resource must be specifically related to **AI agents, multi-agent frameworks, MCP servers, or agent skills** in the context of marketing. We do not accept general AI tools (like ChatGPT wrappers) unless they have autonomous agentic capabilities.
- **Quality:** Ensure the tool or resource is functional, reputable, and provides genuine value to marketers.
- **Formatting:** Use the following format for entries:
  `* [Resource Name](http://link) - A brief, one-sentence description of what the resource does.`
- **Alphabetical Order:** Keep entries in alphabetical order within their respective categories.
- **Search:** Search previous suggestions before making a new one, as yours may be a duplicate.
- **Check your spelling and grammar.**
- **Make sure your text editor is set to remove trailing whitespace.**

Thank you for your suggestions!
